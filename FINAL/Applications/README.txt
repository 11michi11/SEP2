How to run the system applicaitons:

1. Run eNTeServer.jar first. Server application does not have a GUI and in order to see text output,
 run application from console.
To do so, execute the following command in the folder with eNTeServer.jar:
java -jar eNTeServer.jar

2. Run client application - eNTe.jar, by double-clicking on it. GUI of application should appear.
